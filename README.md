# Dotfiles

My macOS / Windows dotfiles.

## Setup

### macOS

Install:  
```bash
$ git clone https://github.com/uhooi/dotfiles.git
$ cd dotfiles
$ sh Install.sh
```

Uninstall:  
```bash
$ sh Uninstall.sh
```

### Windows

Install:  
```bat
$ git clone https://github.com/uhooi/dotfiles.git
$ cd dotfiles
$ start Install.bat
```

Uninstall:  
```bat
$ start Uninstall.bat
```

## Caution

Please take a backup of dotfiles before uninstall.
